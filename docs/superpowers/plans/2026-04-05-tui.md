# DoH Proxy TUI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `doh-proxy-tui` binary — a full-screen interactive terminal UI with Dashboard, Config, Stats, and Query Log tabs — that reads/writes the existing TOML config file and is released alongside the existing CLI binary.

**Architecture:** `ratatui` + `crossterm` render a tabbed full-screen TUI in the alternate terminal buffer. A crossterm event thread feeds key events and periodic ticks over an `mpsc` channel to a single-threaded async event loop that owns all state. The proxy server runs on a background Tokio thread, sharing an `Arc<Stats>` with the foreground loop. Config changes are validated in memory then written atomically (write-then-rename) to the config file path passed at startup.

**Tech Stack:** Rust 1.77+, `ratatui 0.29`, `crossterm 0.28`, `tokio` (already present), existing `doh_proxy` library crate.

use std::sync::mpsc;
use std::thread;
use std::time::Duration;

use crossterm::event::{self, Event as CtEvent, KeyEvent};

/// Application events delivered to the main loop.
#[derive(Debug)]
pub enum Event {
    /// A key was pressed.
    Key(KeyEvent),
    /// Periodic redraw tick (interval set at construction).
    Tick,
}

/// Spawns a background thread that forwards terminal events and ticks.
pub struct EventHandler {
    rx: mpsc::Receiver<Event>,
}

impl EventHandler {
    /// `tick_rate` — how often `Event::Tick` is emitted when no key arrives.
    pub fn new(tick_rate: Duration) -> Self {
        let (tx, rx) = mpsc::channel();
        thread::spawn(move || {
            loop {
                if event::poll(tick_rate).unwrap_or(false) {
                    if let Ok(CtEvent::Key(key)) = event::read() {
                        if tx.send(Event::Key(key)).is_err() {
                            return; // receiver dropped — exit thread
                        }
                    }
                } else {
                    if tx.send(Event::Tick).is_err() {
                        return;
                    }
                }
            }
        });
        Self { rx }
    }

    /// Block until the next event arrives.
    pub fn next(&self) -> anyhow::Result<Event> {
        self.rx.recv().map_err(|e| anyhow::anyhow!("event channel closed: {e}"))
    }
}

use std::sync::atomic::{AtomicBool, Ordering};
use std::sync::Arc;

use doh_proxy::{config::Config, server::Server, stats::Stats};

use crate::tui::config_editor::ConfigEditor;

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum Tab {
    Dashboard,
    Config,
    Stats,
    Log,
}

impl Tab {
    const ALL: [Tab; 4] = [Tab::Dashboard, Tab::Config, Tab::Stats, Tab::Log];

    pub fn title(self) -> &'static str {
        match self {
            Tab::Dashboard => "Dashboard",
            Tab::Config    => "Config",
            Tab::Stats     => "Stats",
            Tab::Log       => "Query Log",
        }
    }

    pub fn index(self) -> usize {
        Self::ALL.iter().position(|t| *t == self).unwrap()
    }
}

enum ServerHandle {
    Stopped,
    Running {
        stop_flag: Arc<AtomicBool>,
        thread: std::thread::JoinHandle<()>,
    },
}

pub struct AppState {
    pub active_tab: Tab,
    pub config: Config,
    pub config_path: String,
    pub config_editor: ConfigEditor,
    pub stats: Arc<Stats>,
    pub status_msg: Option<String>,
    pub log_scroll: usize,
    server: ServerHandle,
}

impl AppState {
    pub fn new(config_path: String, config: Config) -> Self {
        let config_editor = ConfigEditor::from_config(&config);
        Self {
            active_tab: Tab::Dashboard,
            config_editor,
            config,
            config_path,
            stats: Arc::new(Stats::new()),
            status_msg: None,
            log_scroll: 0,
            server: ServerHandle::Stopped,
        }
    }

    pub fn is_running(&self) -> bool {
        matches!(self.server, ServerHandle::Running { .. })
    }

    pub fn next_tab(&mut self) {
        let idx = (self.active_tab.index() + 1) % Tab::ALL.len();
        self.active_tab = Tab::ALL[idx];
        self.log_scroll = 0;
    }

    pub fn prev_tab(&mut self) {
        let idx = (self.active_tab.index() + Tab::ALL.len() - 1) % Tab::ALL.len();
        self.active_tab = Tab::ALL[idx];
        self.log_scroll = 0;
    }

    pub fn start_server(&mut self) {
        if self.is_running() {
            return;
        }
        let stop_flag = Arc::new(AtomicBool::new(false));
        let config = self.config.clone();
        let stats = Arc::clone(&self.stats);
        let stop_clone = Arc::clone(&stop_flag);

        let thread = std::thread::spawn(move || {
            let rt = tokio::runtime::Runtime::new().expect("tokio runtime");
            rt.block_on(async move {
                match Server::new(config, Some(stats)).await {
                    Ok(server) => {
                        if let Err(e) = server.run_cancellable(stop_clone).await {
                            tracing::error!("server error: {e}");
                        }
                    }
                    Err(e) => tracing::error!("server init error: {e}"),
                }
            });
        });

        self.server = ServerHandle::Running { stop_flag, thread };
        self.status_msg = Some(format!("Server started on {}", self.config.listen_addr));
    }

    pub fn stop_server(&mut self) {
        let old = std::mem::replace(&mut self.server, ServerHandle::Stopped);
        if let ServerHandle::Running { stop_flag, thread } = old {
            stop_flag.store(true, Ordering::Release);
            let _ = thread.join();
        }
        self.status_msg = Some("Server stopped.".into());
    }

    pub fn scroll_log_down(&mut self, total_entries: usize) {
        if self.log_scroll + 1 < total_entries {
            self.log_scroll += 1;
        }
    }

    pub fn scroll_log_up(&mut self) {
        self.log_scroll = self.log_scroll.saturating_sub(1);
    }
}

impl Drop for AppState {
    fn drop(&mut self) {
        // Ensure server is stopped on exit even during a panic unwind.
        self.stop_server();
    }
}

```rust
use std::io::Write;

use doh_proxy::config::{CacheConfig, Config};

/// In-memory form state for the Config tab.
/// All fields are stored as strings for direct keyboard editing.
pub struct ConfigEditor {
    pub listen_addr: String,
    pub upstreams: Vec<String>,
    pub cache_enabled: bool,
    pub cache_capacity: String,
    /// Which field the cursor is on (0 = listen_addr, 1..N = upstreams, N+1 = cache_enabled, N+2 = capacity)
    pub focused: usize,
    /// True when the focused text field is being edited (raw character input mode).
    pub editing: bool,
}

impl ConfigEditor {
    pub fn from_config(config: &Config) -> Self {
        Self {
            listen_addr: config.listen_addr.to_string(),
            upstreams: config.upstreams.clone(),
            cache_enabled: config.cache.enabled,
            cache_capacity: config.cache.capacity.to_string(),
            focused: 0,
            editing: false,
        }
    }

    /// Total number of navigable fields.
    pub fn field_count(&self) -> usize {
        1 + self.upstreams.len() + 2 // listen_addr + upstreams + cache_enabled + cache_capacity
    }

    pub fn focus_next(&mut self) {
        self.editing = false;
        self.focused = (self.focused + 1) % self.field_count();
    }

    pub fn focus_prev(&mut self) {
        self.editing = false;
        let n = self.field_count();
        self.focused = (self.focused + n - 1) % n;
    }

    /// Index of the cache_enabled checkbox field.
    pub fn cache_enabled_idx(&self) -> usize {
        1 + self.upstreams.len()
    }

    /// Index of the cache_capacity text field.
    pub fn cache_capacity_idx(&self) -> usize {
        1 + self.upstreams.len() + 1
    }

    /// Handle a printable character while in editing mode.
    pub fn push_char(&mut self, c: char) {
        if let Some(field) = self.focused_text_field_mut() {
            field.push(c);
        }
    }

    /// Handle backspace while in editing mode.
    pub fn pop_char(&mut self) {
        if let Some(field) = self.focused_text_field_mut() {
            field.pop();
        }
    }

    fn focused_text_field_mut(&mut self) -> Option<&mut String> {
        if self.focused == 0 {
            return Some(&mut self.listen_addr);
        }
        let upstream_start = 1;
        let upstream_end = upstream_start + self.upstreams.len();
        if self.focused >= upstream_start && self.focused < upstream_end {
            return Some(&mut self.upstreams[self.focused - upstream_start]);
        }
        if self.focused == self.cache_capacity_idx() {
            return Some(&mut self.cache_capacity);
        }
        None // cache_enabled is a checkbox, not a text field
    }

    /// Add a blank upstream entry below the currently focused upstream.
    pub fn add_upstream(&mut self) {
        self.upstreams.push("https://".into());
        self.focused = self.upstreams.len(); // focus the new entry
    }

    /// Remove the currently focused upstream, if any.
    pub fn remove_focused_upstream(&mut self) {
        let upstream_start = 1usize;
        let upstream_end = upstream_start + self.upstreams.len();
        if self.focused >= upstream_start && self.focused < upstream_end && self.upstreams.len() > 1 {
            self.upstreams.remove(self.focused - upstream_start);
            if self.focused >= upstream_start + self.upstreams.len() {
                self.focused = self.focused.saturating_sub(1);
            }
        }
    }

    /// Validate the form and produce a `Config` on success.
    pub fn validate_and_build(&self) -> anyhow::Result<Config> {
        let listen_addr: std::net::SocketAddr = self
            .listen_addr
            .trim()
            .parse()
            .map_err(|e| anyhow::anyhow!("Invalid listen address: {e}"))?;

        let upstreams: Vec<String> = self
            .upstreams
            .iter()
            .map(|u| u.trim().to_string())
            .filter(|u| !u.is_empty())
            .collect();

        if upstreams.is_empty() {
            anyhow::bail!("At least one upstream URL is required");
        }

        for url in &upstreams {
            if !url.starts_with("https://") && !url.starts_with("http://") {
                anyhow::bail!("Upstream must start with https:// or http://: {url}");
            }
        }

        let capacity: u64 = self
            .cache_capacity
            .trim()
            .parse()
            .map_err(|e| anyhow::anyhow!("Invalid cache capacity: {e}"))?;

        Ok(Config {
            listen_addr,
            upstreams,
            cache: CacheConfig {
                enabled: self.cache_enabled,
                capacity,
            },
        })
    }

    /// Validate, then atomically write TOML to `path` (write tmp → rename).
    pub fn save(&mut self, path: &str) -> anyhow::Result<()> {
        let config = self.validate_and_build()?;

        let toml = format!(
            "listen_addr = \"{}\"\n\nupstreams = [\n{}]\n\n[cache]\nenabled = {}\ncapacity = {}\n",
            toml_escape(&config.listen_addr.to_string()),
            config
                .upstreams
                .iter()
                .map(|u| format!("    \"{}\",\n", toml_escape(u)))
                .collect::<String>(),
            config.cache.enabled,
            config.cache.capacity,
        );

        let tmp_path = format!("{path}.tmp");
        {
            let mut f = std::fs::File::create(&tmp_path)
                .map_err(|e| anyhow::anyhow!("Cannot write config: {e}"))?;
            f.write_all(toml.as_bytes())
                .map_err(|e| anyhow::anyhow!("Cannot write config: {e}"))?;
        }
        std::fs::rename(&tmp_path, path)
            .map_err(|e| anyhow::anyhow!("Cannot replace config file: {e}"))?;

        Ok(())
    }
}

fn toml_escape(s: &str) -> String {
    s.replace('\\', "\\\\").replace('"', "\\\"")
}

use ratatui::{

use crate::tui::app::AppState;

pub fn render(frame: &mut Frame, area: Rect, app: &AppState) {
    let ed = &app.config_editor;

    let mut rows: Vec<(&str, String, bool)> = Vec::new(); // (label, value, is_focused)
    rows.push(("Listen Address", ed.listen_addr.clone(), ed.focused == 0));

    for (i, u) in ed.upstreams.iter().enumerate() {
        let label = if i == 0 { "Upstream 1" } else { format!("Upstream {}", i + 1).leak() };
        rows.push((label, u.clone(), ed.focused == 1 + i));
    }

    let cache_en_idx = ed.cache_enabled_idx();
    let cache_cap_idx = ed.cache_capacity_idx();

    rows.push((
        "Cache Enabled",
        if ed.cache_enabled { "yes" } else { "no" }.into(),
        ed.focused == cache_en_idx,
    ));
    rows.push(("Cache Capacity", ed.cache_capacity.clone(), ed.focused == cache_cap_idx));

    let constraints: Vec<Constraint> = rows
        .iter()
        .map(|_| Constraint::Length(3))
        .chain(std::iter::once(Constraint::Min(0)))
        .collect();

    let chunks = Layout::default()
        .direction(Direction::Vertical)
        .margin(1)
        .constraints(constraints)
        .split(area);

    for (i, (label, value, focused)) in rows.iter().enumerate() {
        let display = if *focused && ed.editing {
            format!("{value}█") // show cursor
        } else {
            value.clone()
        };

        let style = if *focused {
            Style::default().fg(Color::Yellow).add_modifier(Modifier::BOLD)
        } else {
            Style::default()
        };

        let border_style = if *focused {
            Style::default().fg(Color::Yellow)
        } else {
            Style::default()
        };

        let widget = Paragraph::new(Line::from(vec![Span::styled(display, style)]))
            .block(
                Block::default()
                    .borders(Borders::ALL)
                    .border_style(border_style)
                    .title(format!(" {label} ")),
            );
        frame.render_widget(widget, chunks[i]);
    }

    // Hint row at the bottom
    let hints = if app.config_editor.editing {
        "Enter/Esc: finish  Backspace: delete"
    } else if app.is_running() {
        "Stop the server before editing config"
    } else {
        "Tab/↑↓: navigate  Enter: edit  a: add upstream  d: del upstream  F2: save"
    };
    let hint = Paragraph::new(hints).style(Style::default().fg(Color::DarkGray));
    if let Some(hint_area) = chunks.last() {
        frame.render_widget(hint, *hint_area);
    }
}

/// Handle a key event on the Config tab. Returns an optional status message.
pub fn handle_key(app: &mut AppState, key: KeyEvent) -> Option<String> {
    if app.is_running() {
        return Some("Stop the server before editing config.".into());
    }

    let ed = &mut app.config_editor;

    if ed.editing {
        match key.code {
            KeyCode::Esc | KeyCode::Enter => {
                ed.editing = false;
            }
            KeyCode::Backspace => ed.pop_char(),
            KeyCode::Char(c) => ed.push_char(c),
            _ => {}
        }
        return None;
    }

    // Toggle checkbox when focused on cache_enabled
    if ed.focused == ed.cache_enabled_idx() {
        if matches!(key.code, KeyCode::Enter | KeyCode::Char(' ')) {
            ed.cache_enabled = !ed.cache_enabled;
            return None;
        }
    }

    match key.code {
        KeyCode::Tab | KeyCode::Down => ed.focus_next(),
        KeyCode::BackTab | KeyCode::Up => ed.focus_prev(),
        KeyCode::Enter => {
            // Only enter editing mode for text fields (not checkbox)
            if ed.focused != ed.cache_enabled_idx() {
                ed.editing = true;
            }
        }
        KeyCode::Char('a') => ed.add_upstream(),
        KeyCode::Char('d') => ed.remove_focused_upstream(),
        KeyCode::F(2) => {
            match ed.save(&app.config_path.clone()) {
                Ok(()) => {
                    if let Ok(new_cfg) = ed.validate_and_build() {
                        app.config = new_cfg;
                    }
                    return Some("Config saved successfully.".into());
                }
                Err(e) => return Some(format!("Save failed: {e}")),
            }
        }
        _ => {}
    }
    None
}

```rust
use std::io;
use std::time::Duration;

use clap::Parser;
use crossterm::{
    event::{DisableMouseCapture, EnableMouseCapture, KeyCode, KeyModifiers},
    execute,
    terminal::{disable_raw_mode, enable_raw_mode, EnterAlternateScreen, LeaveAlternateScreen},
};
use ratatui::{backend::CrosstermBackend, Terminal};

use doh_proxy::config::Config;
use doh_proxy::tui::{
    app::AppState,
    event::{Event, EventHandler},
    tabs::config as config_tab,
    ui,
};

#[derive(Parser)]
#[command(name = "doh-proxy-tui", about = "Interactive terminal UI for DoH Proxy")]
struct Cli {
    /// Path to config file
    #[arg(short, long, default_value = "config/default.toml")]
    config: String,
}

fn main() -> anyhow::Result<()> {
    let cli = Cli::parse();

    let config = Config::from_file(&cli.config).unwrap_or_else(|_| {
        eprintln!("Warning: could not read {:?}, using defaults", cli.config);
        Config::default()
    });

    // Setup terminal
    enable_raw_mode()?;
    let mut stdout = io::stdout();
    execute!(stdout, EnterAlternateScreen, EnableMouseCapture)?;
    let backend = CrosstermBackend::new(stdout);
    let mut terminal = Terminal::new(backend)?;

    // Run — always restore terminal on exit, even on panic.
    let result = run(&mut terminal, cli.config, config);

    // Restore terminal
    disable_raw_mode()?;
    execute!(
        terminal.backend_mut(),
        LeaveAlternateScreen,
        DisableMouseCapture
    )?;
    terminal.show_cursor()?;

    result
}

fn run(
    terminal: &mut Terminal<CrosstermBackend<io::Stdout>>,
    config_path: String,
    config: Config,
) -> anyhow::Result<()> {
    let mut app = AppState::new(config_path, config);
    let events = EventHandler::new(Duration::from_millis(250));

    loop {
        terminal.draw(|frame| ui::render(frame, &app))?;

        match events.next()? {
            Event::Tick => {
                // Clear one-shot status messages after a few ticks.
                // (We track ticks via a simple counter on AppState if needed;
                // for now status messages persist until next action.)
            }
            Event::Key(key) => {
                // Global quit
                if key.code == KeyCode::Char('q')
                    || (key.code == KeyCode::Char('c')
                        && key.modifiers.contains(KeyModifiers::CONTROL))
                {
                    app.stop_server();
                    break;
                }

                // Global tab navigation (only when not in editing mode)
                if !app.config_editor.editing {
                    match key.code {
                        KeyCode::Tab => {
                            app.next_tab();
                            app.status_msg = None;
                            continue;
                        }
                        KeyCode::BackTab => {
                            app.prev_tab();
                            app.status_msg = None;
                            continue;
                        }
                        KeyCode::Char('1') => { app.active_tab = doh_proxy::tui::app::Tab::Dashboard; continue; }
                        KeyCode::Char('2') => { app.active_tab = doh_proxy::tui::app::Tab::Config;    continue; }
                        KeyCode::Char('3') => { app.active_tab = doh_proxy::tui::app::Tab::Stats;     continue; }
                        KeyCode::Char('4') => { app.active_tab = doh_proxy::tui::app::Tab::Log;       continue; }
                        _ => {}
                    }
                }

                use doh_proxy::tui::app::Tab;
                match app.active_tab {
                    Tab::Dashboard => match key.code {
                        KeyCode::Char('s') => {
                            if app.is_running() {
                                app.stop_server();
                            } else {
                                app.start_server();
                            }
                        }
                        _ => {}
                    },
                    Tab::Config => {
                        if let Some(msg) = config_tab::handle_key(&mut app, key) {
                            app.status_msg = Some(msg);
                        }
                    }
                    Tab::Stats => {} // read-only, auto-refreshes via Tick
                    Tab::Log => match key.code {
                        KeyCode::Char('j') | KeyCode::Down => {
                            let total = app.stats.snapshot_log().len();
                            app.scroll_log_down(total);
                        }
                        KeyCode::Char('k') | KeyCode::Up => app.scroll_log_up(),
                        _ => {}
                    },
                }
            }
        }
    }

    Ok(())
}

## Task 12: CI workflow, source-only release, and install instructions

GitHub hosts source only. Users install via `cargo install`. This task adds a CI workflow that runs tests on every push/PR, strips the release workflow of binary uploads, and documents the install command in README.

**Files:**
- Create: `.github/workflows/ci.yml`
- Modify: `.github/workflows/release.yml`
- Modify: `README.md`

- [ ] **Step 1: Create CI workflow**

Create `.github/workflows/ci.yml`:

```yaml
name: CI

on:
  push:
    branches: ["master", "main"]
  pull_request:

jobs:
  test:
    name: Test
    runs-on: ${{ matrix.os }}
    strategy:
      matrix:
        os: [ubuntu-latest, macos-latest]
    steps:
      - uses: actions/checkout@v4

      - name: Install Rust
        uses: dtolnay/rust-toolchain@stable

      - name: Install Linux system dependencies
        if: matrix.os == 'ubuntu-latest'
        run: |
          sudo apt-get update
          sudo apt-get install -y libssl-dev pkg-config

      - name: Run tests
        run: cargo test --bin doh-proxy --bin doh-proxy-tui --lib 2>&1

      - name: Check clippy
        run: cargo clippy --bin doh-proxy-tui -- -D warnings 2>&1
```

- [ ] **Step 2: Simplify release.yml to source-only**

Replace `.github/workflows/release.yml` with:

```yaml
name: Release

on:
  push:
    tags:
      - 'v*'

jobs:
  release:
    name: Create release
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Install Rust
        uses: dtolnay/rust-toolchain@stable

      - name: Install system dependencies
        run: |
          sudo apt-get update
          sudo apt-get install -y libssl-dev pkg-config

      - name: Run tests
        run: cargo test --bin doh-proxy --bin doh-proxy-tui --lib 2>&1

      - name: Create GitHub release
        uses: softprops/action-gh-release@v2
        with:
          generate_release_notes: true
          body: |
            ## Install

            Requires [Rust](https://rustup.rs) 1.77+.

            ```bash
            cargo install --git https://github.com/${{ github.repository }} --tag ${{ github.ref_name }} --bin doh-proxy-tui
            ```

            Or install the headless proxy daemon:

            ```bash
            cargo install --git https://github.com/${{ github.repository }} --tag ${{ github.ref_name }} --bin doh-proxy
            ```
```
